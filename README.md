# VerseKeeper

VerseKeeper is an intelligent Bible memorization application that helps users memorize Scripture through adaptive spaced repetition, curated verse collections, and multiple review modes.

## Features

- 📖 Curated verse collections
- 🧠 Adaptive spaced repetition
- ⏱️ Daily study sessions based on available time
- ✍️ Exact typing practice
- 🧩 Multiple review modes
- 📊 Progress tracking
- 📱 Responsive design

## Tech Stack

- React
- TypeScript
- Django
- Django REST Framework
- PostgreSQL
